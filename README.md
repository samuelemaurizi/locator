# Locator

### Save your favorite locations.

Techs:

- Node
- Express
- MongoDB
- GeoJSON
- Mapbox Library
- Node-geocoder (library for geocoding)

### Quick Start

Add your MONGO_URI and GEOCODER_API_KEY to the "config/config.env" file.

```
### Install dependencies
npm install

### Serve on localhost:5000
npm run dev (nodemon)
or
npm start
```
